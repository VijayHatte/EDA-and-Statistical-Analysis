# EDA and Statistical Analysis

This repository contains structured notebooks covering Exploratory Data Analysis (EDA), Pandas data manipulation, Matplotlib visualization, and various Statistical/Mathematical concepts.

## 📂 Project Structure

```text
├── .gitignore
├── README.md
├── requirements.txt           # Python dependencies
├── data/                      # Place dataset CSV files here
└── notebooks/                 # Organized Jupyter Notebooks
    ├── eda/                   # Case studies (Netflix, Pollution, Treadmill)
    ├── matplotlib/            # Tutorials on plotting
    ├── pandas/                # Data cleaning and manipulation tutorials
    └── stat-math/             # Statistics, distributions, and hypothesis tests
```

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VijayHatte/EDA-and-Statistical-Analysis.git
   cd EDA-and-Statistical-Analysis
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Provide Datasets:**
   Since dataset files are large and local, copy the following files from your local downloads into the `data/` folder of this repository:
   * `netflix_titles_2021.csv`
   * `cities_air_quality_water_pollution.18-10-2021 (1).csv`
   * `CardioGoodFitness.csv`
   * `AmesHousing.csv`
   * `winemag-data-130k-v2.csv`
   * `bhp.csv`
   * `heights.csv`
   * `income (1).csv`
   * `income.csv`
   * `movie_revenues.csv`
   * `nyc_weather.csv`
   * `revenue.csv`
   * `stock_data.csv`
   * `vij.csv`
   * `weather.csv`
   * `weather2.csv`
   * `weather3.csv`
   * `weather_by_cities.csv`
   * `weather_data.csv`

4. **Launch Jupyter:**
   ```bash
   jupyter notebook
   ```
